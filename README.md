# 🚀 Portfólio Pessoal — Augusto B. Chupernate

Este projeto foi desenvolvido como parte do **Programa WiseStart 2026**, com o objetivo de criar um **site de portfólio profissional** para apresentação de habilidades, projetos e experiências.

---

## 🧠 Sobre o Projeto

A proposta consiste em desenvolver uma página pessoal moderna, responsiva e visualmente atrativa, que funcione como um cartão de visitas digital para desenvolvedores.

O site apresenta:
- 👤 Informações pessoais
- 💻 Tecnologias dominadas
- 📂 Projetos desenvolvidos
- 📬 Formas de contato

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura da página  
- **CSS3** — Estilização e layout  
- **JavaScript** — Interatividade e animações  
- **Git & GitHub** — Versionamento de código  

---

## 📋 Requisitos do Desafio

### ✅ Obrigatórios
- Estruturação da página utilizando **HTML5**
- Estilização com **CSS**
- Versionamento com **Git/GitHub**

### ⭐ Diferenciais (Implementados)
- ✔️ Responsividade (Mobile, Tablet e Desktop)  
- ✔️ Animações e efeitos visuais  
- ✔️ Menu interativo para mobile  
- ✔️ Alternância entre tema **claro/escuro**  
- ✔️ Animações com JavaScript (typing effect, scroll, etc.)

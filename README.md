# Desafio de Projeto para desenvolver um **Site de Portfólio**

Um dos desafios do **Programa WiseStart 2026**!

Desafio este que consiste em desenvolver um site de portfólio com as seguintes **tecnologias**:
- HTML
- CSS

# Informações detalhadas do desafio:

## Projeto 1: Desenvolver uma Página Pessoal para Apresentação Profissional

Criar uma página pessoal que sirva como portfólio profissional, destacando habilidades, projetos e
experiências do desenvolvedor.

### Requisitos Técnicos

#### Obrigatórios:
    - Utilizar HTML5 para estruturar o conteúdo (ex.: seção de apresentação, projetos, contatos, etc);

    - Usar CSS para estilizar a página, destacando a identidade visual do desenvolvedor;

    - Versionar o código com Git e manter o repositório no GitHub;
#### Desejáveis:
    - Hospedar a página utilizando o GitHub Pages;

    - Implementar responsividade para suportar diferentes tamanhos de tela (ex.: mobile, tablet e desktop);

    - Incluir seções adicionais, como:
        - Apresentação pessoal (foto, nome, descrição); 

        - Habilidades técnicas (com ícones ou gráficos);

        - Projetos anteriores (com links para repositórios ou demos);

        - Contato (formulário funcional ou links para redes sociais);

#### Entregáveis:
● Link do repositório no GitHub contendo o código da página;

● Link da página hospedada no GitHub Pages;

#### Critérios de Avaliação:
1. Funcionalidade básica:
    
    ○ A página é navegável e apresenta o conteúdo esperado.
2. Uso de Git e GitHub:
    
    ○ O histórico de commits reflete o progresso do desenvolvimento.
3. Estética:
    
    ○ A página é visualmente atrativa e organizada.
4. Responsividade:
    
    ○ A página se adapta corretamente a diferentes dispositivos.
5. Extras:
    
    ○ Implementação de animações, efeitos ou outras funcionalidades avançadas será considerado um diferencial.
